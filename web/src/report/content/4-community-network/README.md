# 社區網絡的深耕共伴

## 匯聚善力，築起守護生命的韌性防護網絡

慈濟的慈善服務，源自每一個守護社區的自發性行動。「社區是修行的道場。」志工生活於巷弄，熟識與牽掛，讓關懷在每一個話家常與關心的生活場景中自然展開。我們主動走近高風險家庭與弱勢學子，在災害來臨前備妥守護社區的互助力量。

這份守護，為未來提前準備，自 2007 年起，我們著重脫貧助學與人才賦能，透過「新芽獎學金」鼓勵超過 11 萬名學子在逆境學子積極向學。為延續這份勵學精神，我們進一步針對新世代需求開拓適性化的公益培力路徑，以「Young善青年培育方案」聚焦生命價值的引導，陪伴高中及大專學子在人文環境中涵養軟實力；而在「新芽育成專案」則聚焦專業知能的提升，與培養自立的能力，讓孩子在逐夢路上無後顧之憂。這些培力行動不僅是經濟的支持，更是重塑韌性特質。數據顯示，89% 的學子提升了學習動機、90% 感到更有歸屬感，更有 93% 的青年在成長後願意主動協助他人，將受助的感激轉化為承擔的力量，讓善意在社區中持續循環流動。

針對高齡化社會下的「在地安居風險」，我們以「安美專案」落實全人、全家、全社區、全齡、全面及全程的「六全」照顧。2022 至 2024 年間，我們主動為 7,943 戶家庭修繕危險環境，安裝扶手與止滑措施等，改善弱勢長者的居住品質與行動安全。

值得一提的是，在改善硬體環境的同時，我們亦扮演社會安全網前哨站的功能，覺察未被發現、未開口尋求急難的民眾，進而辨識出 844 位處於體系邊緣的隱蔽個案，協助重新連結至長期且穩定的社會支持系統，讓每一位長者都能在地安老、尊嚴生活。

社區的實踐讓我們看見，受助者在生命翻轉後，更願意成為支持他人的力量，促使良善價值在社區中持續累積、擴散並產生共鳴。當災害衝擊來臨，這份平日累積的互信與共善動能，在關鍵時刻即轉化為社區抗災與復原的堅實基礎。

自 2024 年止，我們與公部門協力推動防災韌性，共同培訓 18,359 位防災士，足跡遍布全臺 95% 的縣市；全臺約五分之一的防災士來自慈濟或與合作單位共同培育。其中 93% 的學員在受訓後提升應變能力與信心，成為災難發生時，首波挺身而出、守護社區的關鍵力量。

我們深信，當伸出援手時，社區便多一分互信；當善意匯聚，韌性也在日常點滴中築起守護家園的基石，使社區更為安定，社會更加平安。

## 一、摘要

過去幾年，在高齡化浪潮、社會疏離與生活壓力遽增的影響下，臺灣社會中隱性弱勢族群逐年增加。許多家庭因深陷經濟困頓與照顧負荷、或因教育與資源匱乏，或受交通、語言、資訊與健康限制，成為「求援無門、隱於社會縫隙」的脆弱對象，其潛在需求往往難以被察覺，亦難以獲得及時被接應。

我們秉持著「苦難人的人走不出來，有福的人要走進去」的精神，走進社區最真實的生活場景，從守護長者的居家安全，到青年學子的教育支持，乃至於深植社區的防災韌性，以「長情大愛」的溫度，陪伴串起人與人之間本具的互動網絡。在真誠的互動中，我們不僅喚起社區原有的互助精神與良善本性，更見證了許多「手心向上」的受助者，翻轉為「手心向下」的助人者，讓這份信任在鄰里間溫暖流動著。

以下將分別介紹我們深耕社區的三個行動策略，以及 2022 年至 2024 年影響力評估的重點專案—安美計畫之居家安全改善服務、多元青年培力專案、防災士培訓專案。
