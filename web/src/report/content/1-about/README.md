# 關於本報告書

## 從一念心到全球共善：在地守護到國際影響力實踐

60 年，在邁向下一個世代、承先啟後的關鍵時刻。我們回顧過往，鑑古觀今，秉持「誠正信實」的態度自我檢視，向大眾徵信，並為未來深耕永續的福田。自創立以來，慈濟始終守持「慈、悲、喜、捨」的精神，透過慈善、醫療、教育、人文等四大志業「分工不分心」，深耕在地培力，並即時馳援全球人道需求。我們因應時代轉型，持續回應社會變遷的挑戰，並在鄰里與國際間建構堅韌的信任網絡，編織慈善安全防護網，讓共善力量得以匯聚並延續。

這些行動與成果，歸功於全球志工的長期投入與陪伴、受益對象翻轉生命的真實回饋、捐款者的信任與支持，以及合作夥伴與社會各界的協力共行。正是每一位重要夥伴的護持，讓我們得以洞察隱性需求並提供適切的關懷，更在實踐中共同促成社會的向善力量。

本報告書的目的，希望讓每一位慈濟職志工見證自己所參與的具體貢獻，並透過具透明度與專業的影響力論述，以易於理解的方式，向社會大眾說明：在氣候變遷與社會快速轉型的脈絡下，慈濟如何從日常到災難、從社區到全球，與每一位夥伴攜手驅動社會良善轉型與生命翻轉。我們誠摯邀請更多個人、社群與組織和慈濟攜手，以善啟善，以愛傳愛，共同擴大影響力，讓善與愛的正向循環更加長遠。
