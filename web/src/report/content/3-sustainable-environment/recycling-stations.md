# 環保回收站

## 專案介紹

- **進行時間**：1990 年至今
- **執行區域**：臺灣
- **受益對象**：環境、志工
- **專案規模**：從 1990 年起截至 2024 年，環保受證志工人數 92,520 人，累計援助 1,394,424 件環保毛毯至救災現場。從 2022 年至 2024 年，每年平均營運 278 個同時作為社區關懷據點的環保站，環保站參訪之社會大眾 89,686 人次，進行 227,636 噸資源回收。
- **影響力評估時間區間**：2022 年至 2024 年，母體數量為 92,520 位志工，本次調查依 95% 信心水準，5% 誤差範圍進行問卷發放，共計回收 1,273 位有效樣本進行分析。

## 專案內容與亮點

1990 年慈濟推動環保志業以來，志工以實際行動長期投入資源回收與環境保護，在行動中實踐。

而我們「做環保」的內涵，不僅是將垃圾做分類回收，同時，也讓物命再生，創造出長者社會投入的更高價值，也讓退休長輩締造生命第二春。

環保，並非爭利，而是爭取時間，讓地球崩毀速度緩一些。慈濟的環保站具有七項功能，分別包含：垃圾減量回收、長者健康供餐、醫療輔具提供、弱勢團體照護、環境教育、長者健康促進、長照 C 據點。

## 影響力成果

慈濟的環保站，具備垃圾減量回收、長者健康供餐、醫療輔具提供、弱勢團體照護、環境教育、長者健康促進、長照 C 據點，這裡提供長者延續社會參與的優質場域，讓志工在付出的過程中，轉化成為服務大眾的良能，同時找回身心和諧與生命的喜悅。我們看見有 87% 的環保站志工感受到身體健康有所改善，整體的改善幅度平均提升 77%。對於許多志工而言，走進環保站，不只是付出，更是在日復一日的環保行動中，為自己累積一份健康與穩定。

我們進一步觀察，志工在環保站的參與不僅帶動身體健康改善，也促進心理滿足感與自我效能感上的提升。有 95% 的志工表示心靈滿足感提升，其中，「更能自主選擇要用什麼方式過日子」的感受提升最為明顯，提升幅度達 84%。這顯示，長期參與環保行動，有助於重建長者對生活選擇權與自我角色的掌握感，進而提升其持續社會參與的意願與穩定度。

參與環保站服務後，志工在環境議題的理解深度與實踐能力明顯提升。有 98% 的志工認為自己更清楚如何做環保，並且志工知道為什麼要保護地球的認知程度提升達 92%。這份理解，讓環保不再只是口號，而成為願意落實於生活的選擇。我們發現，在擔任環保站志工後，有 99% 的志工更願意在日常生活中實踐環保行動。在實際行為上，47% 的志工在家中落實垃圾分類與資源回收的頻率明顯提升，並且有超過 99% 的志工大多數時間都能夠做到；在隨手關燈、節約用水等生活細節上，也有 98% 的志工能在大多數時間維持好習慣。

這些影響力成果，反映的不僅是行為層面的改變，更是一段透過持續陪伴與支持，逐步促成參與者角色轉換與自我認同重建的歷程。我們觀察到，當參與者在被尊重與被信任的關係中投入行動，環保實踐更容易轉化為可長期維持的生活習慣，並在日常中穩定延續。

## 受益對象回饋

> 阿邦伯是內門環保站的傳奇志工。他每天騎著掛有上人法相的單車，拿著自製「釣竿」打撈水溝裡的資源，後來更換成重型四輪推車載滿物資。即便手指常被割傷、天氣惡劣，他仍堅持不歇涼、不罷工。他曾說：「參與慈濟，是這輩子最開心、最受用的人生大事。」，他實踐了「做到最後一口氣」的誓言，在 2023 年帶著愛上人、愛慈濟、愛地球、愛環保的歡喜心圓滿此生。
