# 影響力模式

為了守護眾生與地球，實踐簡樸惜福的生活願行，我們投入三大行動策略：**物命延續**、**教育啟發**和**身心提升**。

## （一）物命延續 — 從化廢為寶到馳援全球的大愛

全臺設立的環保站據點落實「精緻回收」，讓廢棄物被妥善分類回收後，賦予廢棄物第二次生命，這場全民參與的愛惜物命行動，累積產出碳效益可降低 341,025 公噸的二氧化碳當量。將循環再利用的材料整裝，再運送至大愛感恩科技，製成人道救援現場所需的賑災物資，以循環再製的環保毛毯為例，已被送至 47 個國家地區提供給有需要的人使用。我們將資源回收流程賦予更高的使命，讓受災者在寒冬中感受暖意，更以此引導國際社會看見環保回收的深遠價值，進而投入延續物命的行列。

## （二）教育啟發 — 從觀念啟蒙到以身作則的實踐

我們深信「清淨在源頭」，將環境教育深植於環保教育站、行動環保教育車及環保 PaGamO 環保防災電競大賽。透過互動體驗與科技翻轉學習，逾 98% 的環保志工與學子顯著提升環保知能。當學生與社區居民的環保觀念被啟發後，逐漸明白舉手之勞的小事（例如：節水、節電等）也能為地球盡一份心力，這促使 99% 的環保志工與 92% 的學生更願意參與環境友善行動，將「環保」轉化為人與大地、與生命的真誠對話，從個人引導至家戶乃至社會組織，將減碳綠生活落實為日常實踐。

## （三）身心提升 — 從在地安老到長照地球的樂活

環保站不僅是資源循環的基地，更是守護長者的社區關懷據點。在志工低頭彎腰做分類回收的同時，我們結合簡易健康促進活動，透過「當志工」與社會互動，編織起彼此關懷的互助網絡。在慈濟，逾半數環保志工年齡超過 60 歲以上，從家裡走入環保站而成為環保志工，這些長者不只免長照，也從被照顧者轉化為大地守護者，長者們在群體支持中找回生命價值，實踐了身心康健且富有意義的樂活人生。

我們以環保行動串聯物命延續、教育啟發與身心提升，讓善的循環從資源、行動到生命價值不斷累積，形成與地球共生息的永續循環。
