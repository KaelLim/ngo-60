# 身心同行的永續環境

## 以慈悲為本，守護眾生與地球，實踐簡樸惜福的生活願行

「低頭『彎腰做環保』，是珍惜物命；抬頭『挺身說環保』，是傳承價值。這群環保志工用行動宣示：我們免長照，還要以無私的大愛長照地球，做環保做到最後一口氣！」

慈濟的環保志業並非一時的熱情，而是一條「日日精進、恆長累積」的願行之路。我們相信環保是日常生活的修持，而非一次性的活動。正因如此，我們長年走入社區與校園，啟發「清淨在源頭」的自覺；從年幼的學子到年長的「老菩薩」，每個人都能發揮良能，用鼓掌的雙手守護地球。

在全臺超過一萬處的環保站與社區環保點，我們築起延續物命、落實循環經濟的實踐基地。三十餘年來，這份持之以恆的日常實踐，已累積超過 250 萬公噸的回收成果，其產出的碳效益相當於超過 8,500 座大安森林公園的一年固碳量。

這份守護大地的行動，也轉化為「共善」的能量。這些回收資源被賦予新生命，近 140 萬件環保毛毯，隨著慈濟國際賑災的足跡，走入全球超過 45 個國家地區，成為災後陪伴與關懷的力量。學子與志工們在「化廢為寶」的過程中親眼見證，自己雙手的點滴付出，竟能跨越國界，成為溫暖災民身心的力量，體現膚慰大地、與地球共生息的永續願行。

環保實踐不僅改變了環境，更淨化了人心。統計顯示，99% 的參與者在投入環保志工後，更願意在日常中持續落實友善環境的生活型態；95% 的環保志工在付出中感受到心靈滿足感的提升。換言之，環保不只是守護地球的方法，更是讓善念在生活中紮根，留給後代子孫最珍貴的資產。

## 一、摘要

在臺灣，守護環境的行動從未缺席。無論是非營利組織、政府部門、企業，或是關心土地與海洋的個人，都在各自的位置上推動減塑、淨灘、生態保育與永續倡議。

然而，環保真正的力量，不在於聲量，而在於深度；不在於一時的熱情，而在於日復一日的堅持。當守護環境成為每日生活中的選擇：減少浪費、力行分類、降低不必要的消耗，在「需要」與「想要」之間多一份自省，再決定如何消費與行動，那份改變，才會真正發生。

慈濟在環境永續的道路上，不僅只是實踐者，更是環保意識的啟蒙者與守護者。我們相信，帶動改變的關鍵在於「說我所做，做我所說」的實踐力。透過全臺一萬一千處環保點與社區網絡串聯，讓「環保」不只是口號，而是內化為生活態度。從「清淨在源頭」的自覺出發，引導大眾在日常中落實簡樸惜福；讓「老菩薩」在低頭彎腰間找到生命的新價值、讓學子在校園裡學會惜物愛物。這不僅是守護大地，更是一場淨化心靈、與地球共生息的共同修行。

我們在永續環境的三個行動策略，以及 2022 年至 2024 年影響力評估的重點專案—環保教育站、行動環保教育車、PaGamO 環保防災勇士 PK 賽。
