# 人道救援到翻轉生命希望

## 安身安心、拔苦予樂。陪伴世界各地受天災人禍所影響的苦難眾生，走向重生

「一方有難，十方馳援」，當災難發生，我們匯集鄰近志工迅速救急，安定人心；在漫長的復原階段，則以持續的陪伴，支持受災民眾重新站穩腳步，進而翻轉人生，成為助人者。人道行動的核心，不只是回應眼前的急難，更在於守護受傷的與人民。因此，我們在救急階段就同步展開培力，引發在地居民自助助人的良能，匯聚重建家園的力量。

從 COVID-19 疫情席捲全球、烏克蘭衝突下的難民援助、土耳其與敘利亞地震，到花蓮光復鄉救災，慈濟人始終秉持「走在最前，做到最後」的精神前往第一線，在 2019 年至 2025 年間，我們援助惠及超過 2,164 萬人次，迅速提供災民即時的民生物資與心靈膚慰，穩定基本生活所需，陪伴他們度過最艱困的時刻。在臺灣的 COVID-19 疫情和烏克蘭戰爭的援助行動中，超過 88% 的職工、志工與流離失所的烏克蘭難民都更能以平靜的心面對突如其來的衝擊；當情緒與身心逐漸穩定，人們才有力量，開始邁向下一步。

然而，人道關懷從不止於災難發生的當下。真正的復原，來自於使生活重新回到軌道，以及讓受災學子重新相信自己仍擁有未來的選擇權；透過陪伴與教育支持，我們協助超過 14,362 人次受災地區學子重建學習信心與自我效能，為其開啟更多未來可能。

當人心穩定，放下隔閡，願意互相支持時，在地的復原便能持續發生。我們發現，在國際青年伴學的互動，以及波蘭的難民社群中，分別有 87% 的青年伴學志工、90% 的學員，以及 93% 的難民，都更願意付出行動助人，促使受助者之間的互助網絡逐漸成形。

每次援助時我們皆會串聯在地社群與國際間的支持力量，至今，人道援助行動已遍及超過 44 個國家，與 17 個聯合國組織平台與 30 個國際人道組織建立合作關係，讓善的行動跨越國界與文化，在災後持續流動，更在受傷的土地上，慢慢種下希望的種子。

當受助者願意伸手翻轉為助人者，復原的動能便從外援轉為在地重生的力量，使希望的大樹，在土地上緩緩扎根。

## 一、摘要

當災難突如其來，許多家庭在一夕之間失去居所、收入和生活依靠，而災後隨之而來的生計斷層、教育中斷與社會不平等，仍持續影響受災地區的復原進程。在動盪的世界局勢中，災難往往來得急促無情，我們願在最黑暗的時刻，成為提燈照路的行願者，秉持「聞聲救苦」的精神出現在最需要的地方。

從災害發生的第一時間投入救援，到災後的復原與長期陪伴，慈濟志工始終站在受難者的身旁，提供即時的救助，更致力於當地社區深耕，協助受災社區建立自我支持的互助網絡。

以下將分別介紹慈濟人道救援的三個行動策略，以及 2022 年至 2024 年影響力評估的重點專案：COVID-19、國際青年伴學計畫、波蘭人道救援專案。
