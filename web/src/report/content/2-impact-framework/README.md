# 影響力主軸

## 摘要

「淨化人心、社會祥和、祈求天下無災難。」是慈濟創辦人證嚴法師的三願，我們以此作為價值基礎，並導入國際影響力評估方法論，試著將專案成果轉化為較能具體說明的社會影響力。我們聚焦於「永續環境、社區網絡與人道救援。」三大影響力主軸，建構多元夥伴協力的互助體系。

**永續環境**，以「用鼓掌的雙手做環保」為起點，深化「清淨在源頭」的自覺，帶動大眾從個人生活轉型走向社會共好的具體實踐，落實友善環境與友善地球的承諾，建立與地球共生息的永續循環。在**社區網絡**上，我們持續深耕「志工社區化」的行動，以敦親睦鄰、守望相助為基礎，從個人與家庭陪伴出發，提升社區應對風險的調適與復原韌性。在**人道救援**領域，我們跨越宗教、政治與國界，秉持「直接、重點、尊重」原則，以大悲心陪伴苦難者，協助災民從緊急援助走向自力更生的安居重建之路。

此三大主軸相互串連，如「粽串」般緊密連結組織的核心價值與執行綱領，將佛法智慧實踐為具體的利他行動。透過長情大愛的恆常陪伴，讓個人得以安身，社區得以安心，世界得以安穩，共同推動深耕具備韌性、利他共融的未來。
