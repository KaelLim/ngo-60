# 專案篩選原則

本報告書在專案選擇上，我們審慎檢視各項專案的性質與特色，並依據議題涵蓋範圍、行動代表性與資料可取得性進行篩選。所納入之專案，不僅回應當前社會與環境所面臨的重要挑戰，也與我們長年秉持的行動精神相契合，同時具備實務上的評估可行性與資料完整度。

然而，本報告所呈現之專案，並非涵蓋所有行動樣態，而是在既定時間區間與研究方法條件下所選取之代表性案例。相關影響力成果，主要反映特定受益群體於調查期間之回饋與變化趨勢，適合作為理解方向與觀察脈絡之參考，而非絕對全面之量化呈現。

透過這樣的篩選與揭露方式，我們期盼所呈現的影響力，不僅是成果的彙整，更是一段記錄持續陪伴與長期投入的歷程，讓社會大眾看見善念如何在不同場域逐步累積，並在時間推移中形塑長遠而穩定的正向力量。
